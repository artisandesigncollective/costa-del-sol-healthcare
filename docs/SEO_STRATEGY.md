# SEO Content Strategy

## Target Audience Analysis

### Primary Demographics
- **Age**: 45-70 years old
- **Origin**: UK (60%), Ireland (15%), US/Canada (15%), Australia/NZ (5%), Other EU (5%)
- **Status**: Retirees, remote workers, property investors, families relocating
- **Income**: Middle to upper-middle class
- **Tech Savviness**: Moderate - use smartphones, Google search, Facebook groups

### Search Behavior
- Use Google primarily (95%+)
- Search in English
- Look for "near me" and location-specific terms
- Value reviews and ratings
- Want clear pricing information
- Expect mobile-friendly sites

## On-Page SEO Checklist

### Meta Tags
- [ ] Unique title tag (50-60 chars) for every page
- [ ] Compelling meta description (150-160 chars)
- [ ] Canonical tags to prevent duplicate content
- [ ] Open Graph tags for social sharing
- [ ] Twitter Card tags

### Headers
- [ ] Single H1 per page with primary keyword
- [ ] H2s for main sections
- [ ] H3s for subsections
- [ ] Logical header hierarchy

### Content
- [ ] Minimum 500 words per main page
- [ ] Keyword in first 100 words
- [ ] Related keywords throughout
- [ ] Internal linking (3-5 per page)
- [ ] External links to authoritative sources

### Images
- [ ] Descriptive file names
- [ ] Alt text with keywords
- [ ] Compressed for fast loading
- [ ] WebP format where possible

### Technical
- [ ] Mobile-responsive design
- [ ] Fast page speed (<3s load time)
- [ ] Schema markup for organizations
- [ ] XML sitemap
- [ ] Robots.txt

## Schema Markup Strategy

### LocalBusiness Schema
For each provider:
```json
{
  "@context": "https://schema.org",
  "@type": "MedicalBusiness",
  "name": "Hospital Costa del Sol",
  "address": {...},
  "geo": {...},
  "telephone": "...",
  "priceRange": "$$",
  "aggregateRating": {...}
}
```

### FAQPage Schema
For FAQ page to enable rich snippets

### BreadcrumbList Schema
For navigation breadcrumbs

### Organization Schema
For the directory itself
